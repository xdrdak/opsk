Features

Packed with stuff!

All the features you expect in a single page design has been included to make your life easier.

Little goodies like :

  • Full screen intro page
  • Slick navigation between sections
  • CSS3 Navigations
  • Awesome contact-us footer (with honeypot)

Bootstrap & Initializr

This kit uses Twitter Bootstrap as the base framework, which makes it easy to customize and edit if ever you're familiar with it.

Got a theme you designed for Bootstrap? Simply copy all the necessary ressources!

It also uses Initializr to make sure you're HTML5 ready!

About

It's for fast production

After a heated debate with one of my collegues on how to properly make a one page design, I've decided to make an easy to use template for future use.

This project is for simplifying the prototyping process and deploy newly created websites as fast as possible so we can get back to doing even more awesome things. Like coding awesome algorithms or drinking more tea.
